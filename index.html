Redact sensitive, confidential and privileged information.

Features



Markup

Users can manually select the content, you can build custom redaction template,
or use built-in or custom text matching patterns to mark areas for redaction.

Redaction Preview

Enables user to preview the redactions on a document or
share a document with redaction annotations with other
stakeholders before changes are applied.



Apply Redaction

Apply redactions programmatically or at the tap of a botton to permanently
and securely remove data from the document

Automating Permanent Data Removal

How it works?

Redaction in document is a two-stage process.

  • First, the content to be removed is identified and marked using Spacy
  • Second, the redaction annotations are applied, permanently removing the content from marked regions
